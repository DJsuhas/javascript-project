# JavaScript Demo Project

This is a simple JavaScript project demonstrating basic interaction between **HTML and JavaScript** using a `script.js` file.

## 📌 Features

* Basic JavaScript functionality
* Button click event handling
* DOM manipulation
* Beginner-friendly project structure

## 📂 Project Structure

```
project-folder
│
├── index.html
├── script.js
└── README.md
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/USERNAME/REPO_NAME.git
```

### 2. Navigate to the Project Folder

```bash
cd REPO_NAME
```

### 3. Run the Project

Open `index.html` in your browser.

## 💻 Technologies Used

* HTML5
* JavaScript (Vanilla JS)

## 📖 How It Works

* The HTML page contains a button and a message.
* When the button is clicked, JavaScript updates the message dynamically using DOM manipulation.
